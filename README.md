# Project Title:
Student Grade Record: Record Student's Grade and Score In a Better and Fast School Access

# Project Description:
The Student Grade Record is a student score tracker which is used to grade students in an easy and fast way. This software is aimed at teachers looking for a more efficient way of grading their students.

# Features:
1. Stores student data in JSON format.
2. Full CRUD (Create, Read, Update, Delete) operations via Flask.
3. Able to calculate highest, lowest, and average grades.
4. User-friendly interface.

# Software Used:
Language: Python
Framework: Flask
Repository: GitHub
